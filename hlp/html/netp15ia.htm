Display Tab

The Display tab contains options to configure the colors displayed in the graphs and icons, which icon to use, and the display mode of the NetPerSec dialog.

Graph & Icon Colors

Sample Icon

Illustrates how the icon will appear in the system tray.

Bar graph and Histogram icon

Selects the type of icon to be displayed in the system tray. See NetPerSec's icon section for more information.

Send and Receiving and Background Colors

Sets the colors used in the system tray icon and in the graphs in the Graph Tab. Clicking on either of these buttons will display a color selection window. Because the system tray only supports 16 colors, the selection of colors is limited.

Average and Graph Background Colors

Clicking on these buttons will display the Windows Color dialog. Since these items are not limited to the 16 colors used in the icon, you may select any color in the available from the color dialog. You must be using High Color or True Color for certain colors to appear correctly.

Undo

Restores all colors to their previously saved values.

Default Colors

Sets all colors to NetPerSec's default colors.

Startup and Display

Start with Windows

Automatically start NetPerSec with each session of Windows.

Always on Top

Displays the NetPerSec dialog on top of all other windows.
